# SportBuddy - Find Sports Partners

A mobile app to help people find sports partners and organize events together.

## Features

- **User Profiles**: Create and manage your profile with favorite sports
- **Create Events**: Organize sports events with details like date, time, location
- **Browse Events**: Find and join sports events near you
- **Event Chat**: Chat with other participants in each event
- **Schedule**: View all events you've joined in one place

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Expo CLI

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on your device:
   - Install Expo Go app on your phone
   - Scan the QR code from the terminal
   
   Or run on emulator:
   - iOS: `npm run ios`
   - Android: `npm run android`
